1. Modelo de Lotka Volterra

Considere el modelo \[ \begin{align} \frac{d x_1(t)}{dt}= &r_1x_1\left[1-\frac{x_1(t)+\alpha_{12}x_2(t)}{K_1}\right]\\ \frac{d x_2(t)}{dt}= &r_2x_2\left[1-\frac{x_2(t)+\alpha_{21}x_1(t)}{K_2}\right] \end{align} \] donde \(x_1\) y \(x_2\) representan dos poblaciones que comparten un ecosistema. Las variables \(x_1\) y \(x_2\) pueden representar los tamaños de las dos poblaciones en términos absolutos.

El modelo representado por las ecuaciones anteriores se denomina modelo de Volterra-Lotka y representa muchas relaciones posibles, según sean los signos de \(\alpha_{12}\) y \(\alpha_{21}\), como se muestra a continuación:

\(\alpha_{12}\) \(\alpha_{21}\) Tipo de relación
\(-\) \(-\) Mutualismo
\(-\) \(0\) Comensal
\(0\) \(-\) Comensal
\(+\) \(-\) Parasitismo
\(-\) \(+\) Parasitismo
\(+\) \(+\) Competencia

Cada grupo debe seleccionar dos de estos casos, escoger unos valores razonables de los parámetros y responder las preguntas a continuación para cada uno de estos casos.

Implementación del modelo:

library(deSolve)
library(ggplot2)
library(gridExtra)

simulate_LV <- function(r1, r2, a12, a21, K1, K2, x1, x2, tmax) {
  params <- c(r1=r1, r2=r2, a12=a12, a21=a21, K1=K1, K2=K2)
  state <- c(x1=x1, x2=x2)
  
  LV <- function(t, state, params) {
    with(as.list(c(state, params)), {
      dx1 = r1 * x1 * (1 - ((x1 + a12 * x2) / K1))
      dx2 = r2 * x2 * (1 - ((x2 + a21 * x1) / K2))
      list(c(dx1, dx2))
    })
  }
  
  times <- seq(0, tmax, by=0.01)
  
  out <- as.data.frame(ode(y = state, times=times, func=LV, parms=params))
  
  ggplot(out, aes(x = time)) +
    geom_line(aes(y = x1, color = "x1")) +
    geom_line(aes(y = x2, color = "x2")) +
    xlab("Time") +
    ylab("Population Size") +
    scale_color_manual(values = c("x1" = "blue", "x2" = "red"))
  
}

2. Modelos de tiempo discreto

En este problema consideramos un modelo de interacción Huesped-Parasitoide, usada con frecuencia en control biológico. Se trata de un modelo de la forma \[ \begin{align*} N_{t+1} & = \frac{\lambda N_t e^{−a P_t}}{1 + d N_t}\\ P_{t+1} &= c N_t(1 − e^{−a Pt}) \end{align*} \]

Este modelo Huesped - Parasitoide, es conocido como Nicholson-Bailey Modificado, para el cual hemos definido

  • \(N_t\): Densidad (número) de huéspedes
  • \(P_t\): Densidad (número) de parásitos
  • \(\lambda\): Tasa de reproducción del huésped
  • \(a\): Área donde el parasitoide busca huéspedes.
  • \(c\): No. promedio de huevos viables en un huésped
  • \(d\): Capacidad de carga del ecosistema para huéspedes

Vamos a analizar el modelo de Nicholson-Bailey y estudiaremos la estabilidad / inestabilidad del punto de equilibrio. La dinámica de las poblaciones está dada por las ecuaciones de Nicholson-Bailey modificadas: \[ \begin{align*} N_{t+1}&=\dfrac{\lambda\,N_t\,e^{-a\,P_t}}{1+d\,N_t}\\[2mm] P_{t+1}&=c\,N_t(1-e^{-a\,P_t}) \end{align*} \]

3. Modelo de epidemia

Por último, vamos a usar un modelo de epidemia (el mismo SIR que vimos en clase) para estudiar algunos aspectos simples asociados con un brote. El modelo usado es (despreciando tiempo de incubación): \[ \begin{align*} \frac{d S(t)}{dt}&= -\beta\, S\, I\\ \frac{d I(t)}{dt}&= \beta\, S\, I-\gamma\,I\\ \frac{d R(t)}{dt}&= \gamma\,I \end{align*} \] donde \(\gamma=1/T\) siendo $T el número promedio de días que una persona es contagiosa. \(\beta\) se calcula a partir de un parámetro llamado número básico de reproducción: \(\beta = R_0/T\). Para efectos de la simulación, es mejor usar valores de proporción de la población; por lo tanto, recomiendo usar como valores iniciales esos números. Suponga que la población es de un millón de personas y el brote comienza con una sola contagiada (\(I(0) = 1/1000000\) es la fracción). Todos los demás (\(S(0) = 999999/1000000\) es la fracción) son suceptibles y \(R(0)=0\).
